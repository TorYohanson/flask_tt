Application created by Alexander D for test task.

When this application starts, it takes .json file from http://media.mongodb.org/ and loads it into MongoDB. On each request to home directory of the application, it sends POST ajax request to backend. Request is been handled with flask, that asks MongoDB for 20 most populated cities, stored in it. After this operation, response with a json, containing data for graph, is send to the client, where nvd3.js transforms it into a visible graphs.
